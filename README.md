# SpectorMedia
Project Title: SpectorMediaProject – Digital Marketing Website

Objective:
To design a responsive, multi-page website for SPECTOR Media, a digital marketing agency, using HTML, CSS, Bootstrap, Tailwind CSS, and JavaScript. The project showcases branding, interactivity, and animation across public and admin-facing pages, reflecting real-world business application and technical proficiency.

Technologies Used:
- HTML5 for semantic structure
- CSS3 for custom styling and transitions
- Bootstrap 5 for responsive layout and UI components
- Tailwind CSS for utility-based styling and hover effects
- JavaScript for form validation and dynamic interactivity

Theme:
The website follows SPECTOR Media’s official branding theme using a blend of black, dark red, and white. Typography, layout, and imagery are curated to reflect a premium, modern identity. All visuals are optimized for laptop screen size and responsive across devices.

Pages Included:
1. **index.html** – Homepage with hero banner, services, contact form, and navigation
2. **admin.html** – Admin dashboard with login form and campaign overview cards

JavaScript Features:
- Contact form validation (name, email, message)
- Admin login validation (username, password)
- Alert feedback for successful submission

CSS Features:
- Smooth animations: fade-in, slide-up, zoom-in, pop-up
- Responsive layout using Bootstrap grid and Tailwind spacing
- Custom styling for buttons, cards, and transitions
